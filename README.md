# 雞蛋戰車 SVG 迷宮戰

這是一個單檔可執行的 HTML / SVG 網頁小遊戲。

## 如何遊玩

1. 解壓縮 ZIP。
2. 直接用瀏覽器開啟 `index.html`。
3. 使用鍵盤控制檸檬黃雞蛋戰車。

## 操作方式

- `W` / `↑`：向上移動
- `S` / `↓`：向下移動
- `A` / `←`：向左移動
- `D` / `→`：向右移動
- `Space`：發射炮彈
- 滑鼠移動：控制炮塔瞄準方向

## 技術特色

- 單一 HTML 檔案
- 不使用 Canvas
- 不使用外部圖片
- 不使用 CDN 或外部套件
- 主要視覺元素皆由 SVG 繪製
- 包含迷宮、磚牆、鋼塊、灌木、雞蛋戰車、炮彈、粒子、爆炸與 UI
